# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-02

### Added
- Initial release of Solaris Config Editor
- Support for reading and modifying PHP configuration files
- Dot notation support for accessing nested arrays
- Raw PHP code support via `setRaw()` and `addRaw()`
- Array operations with `push()` and `pushRaw()`
- Configuration merging capabilities
- PHP 8.2+ support
- Comprehensive documentation
- GitHub Actions CI/CD workflow
- MIT License

### Features
- Preserves formatting and structure of configuration files
- Fluent interface for method chaining
- Exception handling for invalid operations
- Support for all PHP data types
